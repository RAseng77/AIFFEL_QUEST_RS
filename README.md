# AIFFEL Research Quest 🚀

## 👋 소개
안녕하세요! **박성현**입니다.

AIFFEL Research 과정에서 AI/ML을 학습하며 수행한 프로젝트와 퀘스트를 정리한 레포지토리입니다.
실습 코드, 프로젝트 결과물, 그리고 학습 과정에서의 인사이트를 기록하고 있습니다.

## 📚 과정 소개

### 🎯 MainQuest
AIFFEL Research 과정의 핵심 프로젝트들입니다.
- 각 퀘스트는 특정 AI/ML 주제에 대한 심화 학습과 실습으로 구성
- 이론과 실전을 결합한 종합 프로젝트 수행
- **총 5개의 메인 퀘스트**

### 🔬 Exploration
다양한 AI/ML 기술을 탐색하고 실험하는 프로젝트들입니다.
- 딥러닝, 자연어처리, 컴퓨터 비전 등 다양한 분야 탐구
- 실제 데이터를 활용한 문제 해결 경험
- **총 9개의 탐험 프로젝트**

### 🎓 GoingDeeper
심화 학습 과정으로, 특정 분야를 더 깊이 있게 다룹니다.
- 최신 AI 기술과 연구 동향 학습
- 고급 기법 적용 및 성능 최적화
- **총 8개의 심화 프로젝트**
